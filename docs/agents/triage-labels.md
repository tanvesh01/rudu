# Triage Labels

The canonical triage state-machine labels and their mapping to this repo's GitHub labels.

| Canonical role | GitHub label      |
|----------------|-------------------|
| needs-triage   | `needs-triage`    |
| needs-info     | `needs-info`      |
| ready-for-agent| `ready-for-agent` |
| ready-for-human| `ready-for-human` |
| wontfix        | `wontfix`         |

Category labels (applied alongside state labels):

| Canonical role | GitHub label   |
|----------------|----------------|
| bug            | `bug`          |
| enhancement    | `enhancement`  |

Every triaged issue should carry exactly one category label and one state label.
